Backend App:

1. Make sure that MongoDB is installed in your machine.
2. Install node modules:
go to the backend folder in the command line and run the following command:
`npm install`
3. In order to run the backend server, run the following command:
`node app.js`


Frontend App:

1. Install node modules:
go to the frontend folder in the command line and run the following command:
`npm install`
2. In order to run the frontend server, run the following command:
`npm start`
